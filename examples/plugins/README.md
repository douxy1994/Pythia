# Pythia plugin examples

- `echo-translator.pythia`: no-network runtime and protocol check.
- `openai-compatible-translator.pythia`: network permission, secret configuration, and OpenAI-compatible translation.
- `text-preprocessor.pythia`: deterministic local text preprocessing.

Each directory is an unpacked `.pythia` package. Follow
`Docs/PYTHIA_PLUGIN_DEVELOPMENT_GUIDE.md` to validate, test, and package it.
